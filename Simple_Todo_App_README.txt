Simple Todo App with Login

A basic web-based Todo application with a simple login screen. Built using HTML, CSS, and JavaScript.

 Features
- Simple login form (username and password)
- Add new tasks
- View task list
- Delete individual tasks
- Fully client-side (no backend required)

 Technologies Used
- HTML
- CSS
- JavaScript

 How It Works
1. User logs in with a username and password.
2. Upon login, the Todo app interface is displayed.
3. User can:
   - Enter a task in the input field
   - Click Add to save it to the list
   - Click Delete to remove any task

 Getting Started
1. Clone or download this repository.
2. Open index.html in a web browser.
3. Use the login form to enter any credentials.
4. Start adding your tasks!

 File Structure
project-folder/
 index.html         # Main HTML file with login and todo logic

 Note
This app is only for learning/demo purposes. The login does not connect to any server or perform real authentication.
